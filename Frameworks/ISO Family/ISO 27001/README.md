# ISO/IEC 27001 – Information Security Management System (ISMS)

ISO/IEC 27001 is the international standard for establishing, implementing, maintaining, and continually improving an Information Security Management System (ISMS).
It provides a structured, risk-based framework for protecting information assets and ensuring confidentiality, integrity, and availability.

## Purpose
ISO/IEC 27001 helps organizations:
Identify and assess information security risks
Implement appropriate security controls
Establish governance and accountability
Achieve continual improvement
Demonstrate security assurance to stakeholders
Certification to ISO/IEC 27001 is globally recognized as proof of a mature information security program.

## Structure
The standard follows the ISO Harmonized Structure (Clauses 4–10):
Context of the Organization
Leadership
Planning
Support
Operation
Performance Evaluation
Improvement

It also includes Annex A, which provides a reference set of security controls (aligned with ISO/IEC 27002).

## Core Principles
Risk-based approach
Management commitment
Documented governance framework
Control selection through a Statement of Applicability (SoA)
Continuous monitoring and improvement

## Related Standards

- ISO/IEC 27002 – Security control implementation guidance
- ISO/IEC 27005 – Information security risk management
